## 🚀 Projects


---

### 🥇 [Medical Diagnosis Chatbot](https://github.com/AnushkaSharma05/MediBot) · [Live ↗](https://mediscan-ayxg.onrender.com/) — NLP · PyTorch · Flask
> NLP-powered diagnosis chatbot trained on 4,920 samples across **41 diseases and 131 symptoms**. Fuzzy symptom matching and alias mapping handle real-world input variation. Conversation memory accumulates symptoms across turns; confidence thresholding (<75%) triggers follow-up questions. Deployed via Flask with top-3 predictions and color-coded severity levels.

`Python` `PyTorch` `NLP` `Fuzzy Matching` `Flask` — **100% accuracy · F1: 1.0**

---

### 🥈 [AQI Analysis Agent](https://github.com/AnushkaSharma05/) · [Live ↗](https://aqi-analyzer-agent.streamlit.app/) — AI Agent · Claude API · REST APIs
> Fetches live air quality data via REST APIs, computes AQI using the official US EPA formula across **6 pollutant tiers** (PM2.5, PM10, CO), and uses Claude API to generate personalised health recommendations — under 2 seconds, deployed live on Streamlit Cloud.

`Python` `Claude API` `REST APIs` `Streamlit Cloud` — **<2s response · Live in production**

---

### 🥉 [Multi-Disease Predictor](https://github.com/AnushkaSharma05/multi_disease) — Classification · Multi-Dataset
> Detects Diabetes, Heart Disease & Parkinson's from patient inputs — 3 real medical datasets, SVM + Random Forest + XGBoost tuned via GridSearchCV. Multi-page Streamlit app with disease selection, feature scaling, cross-validation, and clinical-grade evaluation metrics.

`Python` `Scikit-learn` `XGBoost` `Streamlit` `GridSearchCV` — **F1: 0.97 · 95% accuracy · 94% recall**
